Database

ERD, core entities, and table structure

Database Overview

GANAP PH uses Supabase PostgreSQL as its cloud-hosted relational database. The schema is centered on user management, event lifecycle operations, registration records, and payment tracking. UUID-based keys and relational constraints are used to maintain data integrity across modules.

Data Integrity Notes

  • UUID identifiers are used to reduce collision risk and support distributed operations.
  • Foreign-key relationships keep event, user, and registration records consistent.
  • Status fields are used for workflow transitions (e.g., pending, confirmed, cancelled).
  • Webhook-driven payment updates keep registration state synchronized with Stripe results.
