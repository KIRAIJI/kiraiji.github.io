Project Overview

Project title

GANAP PH

Project description

GANAP PH is a barangay event registration website that aims to connect the community by providing an easy way to sign up for local events. It solves the difficulty of joining community activities by closing the information gap and raising awareness about barangay events.

Target users

Barangay staff, community organizers, and residents who want to stay informed and register for local events.

Key features and functionalities

  • Authentication with OAuth (Google / Facebook / Single Sign-On)
  • Barangay staff can manage (CRUD) events
  • Barangay staff can manage and view event responses
  • Users can register for available events
  • Validation of submitted registration data
  • Payments for events using third‑party APIs
  • Email notifications for registration and event updates
  • Activity tracking for users and admins
  • Automatic closing of events when seats are full or registration has ended
