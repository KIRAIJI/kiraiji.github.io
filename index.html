Project Overview

What GANAP PH solves and who built it

Problem

Barangays currently manage events using manual processes such as paper lists, physical attendance sheets, and manual payment tracking. This results in:

  • Lost or duplicate registrations
  • Inefficient check-in processes
  • Lack of real-time tracking
  • Poor payment monitoring

Solution

A centralized digital event management system that automates:

  • Event creation and publishing
  • Online registration (free & paid)
  • Secure payment processing via Stripe
  • QR code generation for attendance
  • Real-time updates using WebSockets
